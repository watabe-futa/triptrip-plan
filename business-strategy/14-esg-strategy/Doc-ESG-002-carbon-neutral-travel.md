## Executive Summary

旅行業界は世界のCO2排出量の約8%を占め、気候変動対策において重要なセクターである。特に航空輸送が観光関連排出の約65%を占める中、インバウンド観光を主軸とするTripTripにとって、カーボンニュートラル旅行の推進は事業の持続可能性と競争力に直結する戦略的課題である。本文書では、旅行のカーボンフットプリント計算・可視化機能の設計、低炭素旅行オプション（新幹線、エコホテル、地元体験）の推進戦略、カーボンオフセットプログラムの詳細設計、そしてグリーンマーケティング戦略を包括的に定義する。2030年までに、TripTrip経由の旅行のカーボンニュートラル化率50%達成を目標とし、環境意識の高い旅行者の獲得と、業界におけるサステナビリティリーダーとしてのポジショニングを確立する。

### 2.2 旅程別カーボン計算機能設計

#### 2.2.1 計算ロジックの概要

**旅程全体のカーボンフットプリント計算**

```
総排出量 = Σ(交通排出量) + Σ(宿泊排出量) + Σ(アクティビティ排出量)

【交通排出量】
国際線: 距離(km) × 排出係数 × 座席クラス係数 × RFI
国内線: 距離(km) × 98 g/km
新幹線: 距離(km) × 17 g/km
在来線: 距離(km) × 22 g/km
バス: 距離(km) × 33 g/km
タクシー: 距離(km) × 132 g/km × 1/乗車人数
レンタカー: 距離(km) × 車種係数 × 1/乗車人数

【宿泊排出量】
宿泊排出量 = 泊数 × 施設タイプ係数
※エコ認証施設は係数を20-40%削減

【アクティビティ排出量】
アクティビティ排出量 = Σ(各アクティビティの排出量)
※アクティビティ別にデータベース化
```

#### 2.2.2 計算精度レベル

| レベル | 精度 | 必要データ | ユースケース |
|--------|------|-----------|-------------|
| Level 1（概算） | ±30% | 出発地、目的地、日数 | 計画初期段階 |
| Level 2（標準） | ±15% | 交通手段、宿泊タイプ | 予約検討段階 |
| Level 3（詳細） | ±5% | フライト番号、施設名、アクティビティ | 予約確定後 |

#### 2.2.3 国際線フライト計算の詳細

**国際線排出量計算のアルゴリズム**

```
┌─────────────────────────────────────────────────────────────┐
│           国際線カーボン計算アルゴリズム                     │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  Input:                                                      │
│  - departure_airport: 出発空港コード（IATA）                │
│  - arrival_airport: 到着空港コード（IATA）                  │
│  - cabin_class: 座席クラス                                  │
│  - aircraft_type: 機材（オプション）                        │
│                                                              │
│  Step 1: 距離計算                                            │
│  distance = haversine(dep_lat, dep_lon, arr_lat, arr_lon)   │
│  distance_adjusted = distance × 1.095 （迂回係数）          │
│                                                              │
│  Step 2: 基本排出係数選択                                   │
│  if distance < 1500km:                                       │
│      base_factor = 0.158 kg/km                               │
│  elif distance < 4000km:                                     │
│      base_factor = 0.131 kg/km                               │
│  else:                                                       │
│      base_factor = 0.109 kg/km                               │
│                                                              │
│  Step 3: 座席クラス係数                                     │
│  cabin_factor = {                                            │
│      'economy': 1.0,                                         │
│      'premium_economy': 1.6,                                 │
│      'business': 2.9,                                        │
│      'first': 4.0                                            │
│  }[cabin_class]                                              │
│                                                              │
│  Step 4: 高高度影響（RFI）                                  │
│  rfi_factor = 1.9                                            │
│                                                              │
│  Step 5: 排出量計算                                         │
│  emissions = distance_adjusted × base_factor ×               │
│              cabin_factor × rfi_factor                       │
│                                                              │
│  Output: emissions (kg CO2e)                                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

#### 3.1.2 低炭素交通推奨アルゴリズム

**推奨ロジック**

```
┌─────────────────────────────────────────────────────────────┐
│           低炭素交通推奨アルゴリズム                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  Input:                                                      │
│  - origin: 出発地                                           │
│  - destination: 目的地                                      │
│  - user_preferences: ユーザー設定                           │
│    - time_sensitivity: 時間重視度（1-5）                   │
│    - eco_sensitivity: 環境重視度（1-5）                    │
│    - budget_sensitivity: 価格重視度（1-5）                 │
│                                                              │
│  Step 1: 利用可能な交通手段の取得                           │
│  options = get_transport_options(origin, destination)        │
│                                                              │
│  Step 2: 各オプションのスコア計算                           │
│  for option in options:                                      │
│      time_score = normalize(1 / option.duration)             │
│      eco_score = normalize(1 / option.co2_emissions)         │
│      budget_score = normalize(1 / option.price)              │
│                                                              │
│      total_score = (                                         │
│          time_score × user_preferences.time_sensitivity +    │
│          eco_score × user_preferences.eco_sensitivity +      │
│          budget_score × user_preferences.budget_sensitivity  │
│      ) / sum(sensitivities)                                  │
│                                                              │
│  Step 3: エコボーナスの適用                                 │
│  if option.is_eco_certified:                                 │
│      total_score += 0.1                                      │
│                                                              │
│  Step 4: ランキング・表示                                   │
│  options.sort_by(total_score, descending=True)               │
│  display_with_eco_badge(options where eco_score > threshold) │
│                                                              │
│  Output: Ranked transport options with eco badges            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 5.3 グリーンウォッシング回避

#### 5.3.1 グリーンウォッシングリスクと対策

**リスク領域と対策**

| リスク | 内容 | 対策 |
|--------|------|------|
| 根拠なき主張 | 検証不能なエコ主張 | データに基づく主張のみ |
| 誇大表現 | 実態以上のエコアピール | 控えめで正確な表現 |
| 部分的真実 | 一部だけを強調 | 全体像の開示 |
| 曖昧な用語 | 「エコフレンドリー」等の曖昧表現 | 具体的な基準明示 |
| 見せかけの認証 | 自社作成の「認証」 | 第三者認証の活用 |

#### 5.3.2 コミュニケーションガイドライン

**DO（推奨）**

- 具体的な数値と根拠を示す
- 第三者認証を活用する
- 改善余地を認める（完璧でないことを認める）
- 進捗を継続的に報告する
- ネガティブな情報も開示する

**DON'T（禁止）**

- 根拠なく「環境に優しい」と主張する
- 競合他社を根拠なく批判する
- 不十分な取り組みを過大に見せる
- 誤解を招く画像・表現を使う
- オフセットのみで「カーボンニュートラル」と主張する
