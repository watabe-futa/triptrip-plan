# 資本構造＆ファイナンシング戦略 [Doc-FP-007]

## エグゼクティブサマリー

本文書は、TripTripの最適資本構造とファイナンシング戦略を包括的に策定します。Goldman Sachs/JP Morganレベルの投資銀行分析手法を用いて、エクイティとデットの最適バランス、資金調達手段の選択、株式インセンティブ設計、エグジットに向けた資本構造の進化を詳細に設計します。2024年のシリーズA（8億円）から2027年のシリーズC/Pre-IPO（35億円）まで、累計58億円の資金調達を通じて、加重平均資本コスト（WACC）を12.5%から9.8%に低下させ、企業価値最大化を実現します。

### 主要財務指標サマリー

**結論**: 成長段階に応じた最適資本構造の設計により、資本コストを最小化しつつ財務の柔軟性を確保し、2028年のIPOに向けた財務基盤を構築します。

#### 1.2.2 WACC計算手法

**加重平均資本コスト（WACC）の計算**
```
WACC計算式:
WACC = (E/V) × Re + (D/V) × Rd × (1 - Tc)

Where:
E = 株式時価総額
D = 有利子負債残高
V = E + D（企業価値）
Re = 株主資本コスト
Rd = 負債コスト
Tc = 法人税率

TripTrip 2024年度計算例:
├── E/V = 85%
├── D/V = 15%
├── Re = 14.0%（CAPMで算出）
├── Rd = 3.5%（銀行借入金利）
├── Tc = 30%
└── WACC = 0.85 × 14.0% + 0.15 × 3.5% × 0.7 = 12.27%
```

**株主資本コスト（CAPM）**
```
CAPM計算式:
Re = Rf + β × (Rm - Rf) + Size Premium + Country Premium

TripTrip計算:
├── Rf（リスクフリーレート）= 1.0%（10年国債）
├── β（ベータ）= 1.8（旅行テック/成長株）
├── Rm - Rf（マーケットリスクプレミアム）= 6.0%
├── Size Premium（小型株プレミアム）= 2.0%
├── Country Premium（日本）= 0.0%
└── Re = 1.0% + 1.8 × 6.0% + 2.0% = 13.8% ≒ 14.0%
```

### I. 投資家向けQ&A

```
想定Q&A（資本政策関連）:
┌─────────────────────────────────────────────────────────────┐
│ Q: なぜこの資本構造（デット比率30%）を目標としているのか？    │
│                                                             │
│ A: 3つの理由があります。                                     │
│ 1. WACC最適化: デット活用による税シールド効果で資本コスト低減│
│ 2. 財務柔軟性: 過度なレバレッジを避け、成長投資余力を確保    │
│ 3. 業界標準: 旅行テック業界の平均D/E 0.45に対し保守的水準    │
├─────────────────────────────────────────────────────────────┤
│ Q: 希釈を最小化するための施策は？                            │
│                                                             │
│ A: 以下の施策を実施しています。                              │
│ 1. 適正バリュエーションでの調達（ダウンラウンド回避）        │
│ 2. デットファイナンスの活用（希釈なしの資金調達）            │
│ 3. 補助金・助成金の積極活用（年間¥1.5-2.0億想定）           │
│ 4. 内部留保の活用（黒字化後の成長資金）                      │
├─────────────────────────────────────────────────────────────┤
│ Q: SOプールが20%は大きすぎないか？                           │
│                                                             │
│ A: 成長企業では15-20%が標準的です。                          │
│ 1. 優秀人材獲得のための競争力確保に必要                      │
│ 2. IPO時には10%程度に縮小予定                               │
│ 3. 従業員のモチベーション・リテンションに効果的              │
│ 4. VCからも人材戦略として支持されている水準                  │
├─────────────────────────────────────────────────────────────┤
│ Q: IPO以外のエグジットオプションの検討状況は？               │
│                                                             │
│ A: IPOを第一優先としつつ、代替オプションも検討しています。   │
│ 1. 戦略的M&A: 大手旅行会社、テック企業からの関心あり         │
│ 2. PE Buyout: 事業が安定した場合の選択肢                     │
│ 3. セカンダリー: 一部株主のリクイディティ提供                │
│ IPOが最も株主価値最大化につながると考えています。            │
└─────────────────────────────────────────────────────────────┘
```
