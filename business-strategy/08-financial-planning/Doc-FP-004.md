# 資金調達戦略・バリュエーション [Doc-FP-004]

## エグゼクティブサマリー

### 資金調達戦略の概要

TripTripは、日本インバウンド観光市場における統合プラットフォームとしての地位確立に向け、総額60億円の段階的資金調達を計画しています。Goldman Sachs、JP Morgan、Morgan Stanleyレベルの投資銀行基準に基づくバリュエーション分析により、2027年度における企業価値520億円（Pre-money）、2028年IPO時の時価総額800〜1,000億円を目標としています。

#### 資金調達ハイライト
```
総調達額: 60億円（2023-2027年）
調達ステージ: シード → シリーズA → シリーズB → シリーズC
創業者持分（IPO時）: 35-40%
従業員ストックオプション: 10-12%
投資家持分合計: 48-55%
目標IRR（投資家向け）: 45-65%
```

#### バリュエーション推移
```
シード（2023年）: Pre-money 5億円 → Post-money 7億円
シリーズA（2024年）: Pre-money 25億円 → Post-money 33億円
シリーズB（2025年）: Pre-money 80億円 → Post-money 95億円
シリーズC（2026年）: Pre-money 200億円 → Post-money 235億円
IPO（2028年）: 時価総額 800-1,000億円
```
