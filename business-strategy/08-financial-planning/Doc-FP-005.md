# 収益性への道筋 & ブレークイーブン分析 [Doc-FP-005]

## エグゼクティブサマリー

TripTripの収益性達成戦略は、日本インバウンド観光市場の成長を追い風に、持続可能な黒字化を実現するための明確なロードマップを提示します。Goldman Sachs/JP Morganレベルの分析手法に基づき、現在の赤字体質（2024年EBITDA率-91.4%）から2027年のEBITDA黒字化（同+24.8%）への転換プロセスを詳細にモデリングします。貢献利益率の改善、オペレーティングレバレッジの活用、ユニットエコノミクス最適化を通じ、2025年Q3の月次黒字化、2027年Q4のFCF正転化を達成します。

### 収益性達成ロードマップ

| マイルストーン | 達成時期 | 主要指標 | 達成条件 |
|--------------|----------|----------|----------|
| 貢献利益黒字化 | 2025年Q1 | 貢献利益率 +5% | 粗利率40%、変動費最適化 |
| 月次EBITDA黒字化 | 2025年Q3 | EBITDA ¥50M/月 | 売上¥350M/月、固定費効率化 |
| 四半期黒字化 | 2026年Q2 | EBITDA ¥200M/Q | スケール効果発現 |
| 年間EBITDA黒字化 | 2027年度 | EBITDA ¥3,100M | 粗利率45%、営業効率化 |
| FCF正転化 | 2027年Q4 | FCF ¥500M/Q | CAPEX効率化、運転資本最適化 |

**結論**: 3つの主要レバー（粗利改善、営業効率化、スケールメリット）の複合効果により、2027年度の持続可能な収益性達成は十分に実現可能です。

## 1. はじめに & コンテキスト

### 1.1 収益性達成の目的

#### 1.1.1 収益性が重要な理由

**1. 投資家価値創造**
```
収益性と企業価値の関係:
├── 収益性向上 → マルチプル上昇
│   └─ 赤字企業: EV/Sales 4-6x
│   └─ 黒字企業: EV/Sales 8-15x
├── FCF創出 → 配当・再投資能力
│   └─ 自己資金での成長が可能
│   └─ 希薄化なしの価値創造
└── リスク低減 → 割引率低下
    └─ 資金調達依存からの脱却
    └─ 経営の自由度向上
```

**2. 事業持続可能性**
```
収益性と事業継続性:
├── 外部資金依存からの脱却
│   └─ 市況に左右されない経営
│   └─ 長期的視点での意思決定
├── 投資余力の確保
│   └─ R&D投資の継続
│   └─ M&A機会の追求
└── 人材競争力の維持
    └─ 報酬水準の維持・向上
    └─ 優秀人材の獲得・維持
```

**3. ステークホルダーへの責任**
```
各ステークホルダーへの影響:
├── 株主: 投資リターンの実現
├── 従業員: 雇用の安定、報酬向上
├── 顧客: 持続的なサービス提供
├── パートナー: 長期的な協業関係
└── 社会: 観光産業への貢献継続
```

## 付録

### A. 収益性指標計算式

**主要計算式**
```
収益性指標の定義:
├── 粗利率 = (売上高 - COGS) / 売上高
├── 貢献利益率 = (売上高 - 変動費) / 売上高
├── EBITDA率 = EBITDA / 売上高
├── 営業利益率 = 営業利益 / 売上高
├── FCF率 = FCF / 売上高
├── DOL = 貢献利益 / 営業利益（営業レバレッジ）
├── ブレークイーブン売上 = 固定費 / 貢献利益率
└── 安全余裕率 = (実績売上 - BEP売上) / 実績売上
```

### C. 用語集

**収益性関連用語**
```
EBITDA: 利払い前・税引き前・減価償却前利益
FCF: フリーキャッシュフロー
DOL: Degree of Operating Leverage（営業レバレッジ）
貢献利益: 売上高 - 変動費
貢献利益率: 貢献利益 / 売上高
ブレークイーブン: 損益分岐点
安全余裕率: 実績売上がBEPを上回る比率
ARPU: Average Revenue Per User（顧客平均単価）
CAC: Customer Acquisition Cost（顧客獲得コスト）
LTV: Lifetime Value（顧客生涯価値）
```
