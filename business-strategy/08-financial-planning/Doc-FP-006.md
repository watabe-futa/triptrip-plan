# コスト構造＆営業レバレッジ [Doc-FP-006]

## エグゼクティブサマリー

本文書は、TripTripのコスト構造を包括的に分析し、営業レバレッジを活用した収益性加速戦略を策定します。Goldman Sachs/JP Morganレベルの分析的厳密さで、固定費・変動費構造、主要コストドライバー、規模の経済効果、営業レバレッジ機会を詳細にモデリングします。現在の総コスト12.8億円（2024年度）から2027年度には85.7億円へ拡大しますが、売上高コスト比率は76.8%から68.6%へ改善し、営業レバレッジ効果により収益性を大幅に向上させます。

### 主要財務指標サマリー

| 指標 | 2024年 | 2025年 | 2026年 | 2027年 | 改善率 |
|------|--------|--------|--------|--------|--------|
| 総コスト | ¥12.8億 | ¥28.5億 | ¥52.3億 | ¥85.7億 | - |
| 売上高コスト比率 | 76.8% | 73.2% | 70.5% | 68.6% | -8.2pt |
| 固定費比率 | 42.5% | 38.8% | 35.2% | 32.0% | -10.5pt |
| 変動費比率 | 57.5% | 61.2% | 64.8% | 68.0% | +10.5pt |
| 営業レバレッジ倍率 | 2.8x | 3.2x | 3.6x | 4.0x | +1.2x |
| 損益分岐点売上高 | ¥9.2億 | ¥16.5億 | ¥25.8億 | ¥38.2億 | - |

**結論**: コスト構造の最適化と営業レバレッジの活用により、売上成長を上回るペースでの利益成長を実現し、2027年度にはEBITDAマージン24.8%を達成します。

#### 1.2.2 営業レバレッジ分析手法

**営業レバレッジ度（DOL: Degree of Operating Leverage）**
```
DOL計算式:
DOL = 貢献利益 / 営業利益
    = (売上高 - 変動費) / (売上高 - 変動費 - 固定費)

例: TripTrip 2024年度
売上高: ¥16.7億
変動費: ¥7.4億（変動費率44.3%）
貢献利益: ¥9.3億
固定費: ¥5.4億
営業利益: ¥3.9億

DOL = ¥9.3億 / ¥3.9億 = 2.38x

意味: 売上が10%増加すると、営業利益は23.8%増加
```

**損益分岐点分析（Break-Even Analysis）**
```
損益分岐点計算:
BEP（売上高）= 固定費 / 貢献利益率
             = 固定費 / (1 - 変動費率)

安全余裕率 = (実売上高 - BEP) / 実売上高 × 100%

感度分析:
- 固定費 ±10% → BEP変動
- 変動費率 ±5% → BEP変動
- 売上単価 ±10% → BEP変動
```
