## Executive Summary

TripTripは、日本を拠点とするインバウンド観光プラットフォームとして、日本国内法規制に加え、グローバル展開に伴い各国・地域の規制に準拠する必要がある。本文書は、旅行業法、データプライバシー規制、決済・金融規制、消費者保護規制の4つのカテゴリにおける包括的なコンプライアンスチェックリストを提供する。各規制について、適用範囲、主要要件、準拠状況、担当部門、対応期限を明確化し、継続的なコンプライアンス管理の基盤とする。本チェックリストは四半期ごとに更新され、法務部門による定期監査の基準文書として機能する。

#### 2.2.2 主要要件

**パッケージ旅行指令の主要要件**

```
┌─────────────────────────────────────────────────────────────┐
│           EU パッケージ旅行指令 主要要件                     │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【パッケージの定義】                                        │
│  以下の少なくとも2つ以上を組み合わせた旅行サービス:        │

│  ・宿泊                                                      │
│  ・レンタカー                                                │

│  4. 履行責任                                                │
│     ・パッケージ全体の履行責任                              │
│     ・不適合時の是正、代替サービス、補償                   │
│                                                              │
│  5. 支払保護（倒産保護）                                    │
│     ・前払い金の保護措置（保険、保証金等）                 │
│     ・本国送還の保証                                        │
```
