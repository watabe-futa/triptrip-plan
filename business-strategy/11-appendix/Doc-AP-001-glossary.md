# TripTrip事業計画 用語集・略語集 [Doc-AP-001]

## Executive Summary

本用語集は、TripTrip事業計画書およびIT計画書で使用される専門用語、略語、技術用語を包括的に定義・解説する参照文書である。旅行業界用語、テクノロジー用語、ビジネス・財務用語、およびTripTrip固有の用語を網羅し、ステークホルダー間での共通理解を促進する。本文書は、戦略文書の読解支援、新規参画メンバーのオンボーディング、外部パートナーとのコミュニケーション効率化を目的として編纂された。各用語には日英対訳を付し、グローバル展開を見据えた国際的なビジネスコミュニケーションにも対応している。

---

# 第1章 はじめに

## 1.1 目的とスコープ

### 1.1.1 本用語集の目的

本用語集は、TripTrip事業計画書群（20万行以上）において使用される専門用語の統一的な定義を提供することを主目的とする。具体的には以下の目的を達成する。

**主要目的**:
1. **共通言語の確立**: ビジネス、技術、財務の各領域にまたがる専門用語の統一定義
2. **コミュニケーション効率化**: ステークホルダー間での誤解防止と議論の効率化
3. **オンボーディング支援**: 新規参画メンバーの学習曲線短縮
4. **外部連携円滑化**: 投資家、パートナー、規制当局との対話における用語統一
5. **品質管理**: 文書作成時の用語使用一貫性確保

### 1.1.2 対象読者

本用語集の想定読者は以下の通りである。

| 読者カテゴリ | 活用シーン | 推奨参照セクション |
|------------|----------|------------------|
| 経営陣・役員 | 戦略意思決定、投資家説明 | ビジネス・財務用語、旅行業界用語 |
| 事業開発担当 | パートナーシップ交渉、新規事業検討 | 旅行業界用語、ビジネス用語 |
| エンジニア | システム設計、技術文書作成 | テクノロジー用語、TripTrip固有用語 |
| プロダクトマネージャー | 要件定義、優先順位付け | 全セクション |
| マーケティング担当 | 施策立案、レポーティング | マーケティング用語、財務用語 |
| 投資家・VC | デューデリジェンス、バリュエーション | 財務用語、ビジネス用語 |
| 外部パートナー | 連携協議、契約交渉 | 旅行業界用語、技術用語 |

### 1.1.3 スコープ定義

**収録範囲**:
- 旅行・観光業界の専門用語（OTA、GDS、ホテル、航空等）
- 情報技術・ソフトウェア開発用語（アーキテクチャ、開発手法、インフラ等）
- ビジネス・経営用語（戦略フレームワーク、組織論等）
- 財務・会計用語（財務諸表、投資指標、バリュエーション等）
- マーケティング用語（デジタルマーケティング、グロースハック等）
- TripTrip固有の概念・用語

**除外範囲**:
- 一般的なビジネス英語（Meeting、Email等）
- 基礎的なIT用語（PC、Internet等）
- 特定技術の詳細実装用語（ライブラリ固有API等）

## 1.3 参照ガイド

### 1.3.1 効率的な参照方法

**アルファベット順索引の活用**:
各セクション内の用語はアルファベット順（A-Z）で配列されている。日本語用語も英語表記を基準にソートされているため、英語名が不明な場合は第6章の索引を参照されたい。

**クロスリファレンスの利用**:
各用語エントリには関連用語が記載されている。概念理解を深めるため、関連用語を連続して参照することを推奨する。

**文書間参照**:
用語の詳細な使用例や戦略的位置づけについては、参照文書欄に記載された計画書文書を確認されたい。

### 1.3.2 更新・改訂ポリシー

本用語集は以下のタイミングで更新される。

| 更新トリガー | 更新内容 | 承認プロセス |
|-------------|---------|-------------|
| 新規用語の出現 | 用語追加 | 文書管理者承認 |
| 業界標準の変更 | 定義修正 | 関連部門レビュー |
| TripTrip戦略変更 | 固有用語更新 | 経営会議承認 |
| 四半期レビュー | 全体見直し | 品質管理チーム |

# 第2章 旅行業界用語（A-Z）

## 2.1 予約・流通関連用語

### A

【ADR】（英語: Average Daily Rate）
- 読み: えーでぃーあーる / 平均客室単価
- 略語: ADR
- カテゴリ: 宿泊業界・収益指標
- 定義: 一定期間における販売客室の平均単価。総客室売上を販売客室数で除して算出する。
- 計算式: ADR = 総客室売上 ÷ 販売客室数
- 文脈: TripTripのホテル予約サービスにおける収益性分析、価格戦略立案に使用。
- 関連用語: RevPAR、OCC、GOPPAR
- 参照文書: Doc-FM-002（財務モデル）、Doc-MA-003（市場セグメンテーション）

【Affiliate Marketing】（英語: Affiliate Marketing）
- 読み: あふぃりえいとまーけてぃんぐ / アフィリエイトマーケティング
- カテゴリ: 予約・流通
- 定義: 成果報酬型の広告手法。提携パートナー（アフィリエイター）が自社サービスを紹介し、成約に応じて報酬を支払う仕組み。
- 文脈: TripTripの顧客獲得チャネル戦略における重要施策。旅行ブログ、インフルエンサーとの連携。
- 関連用語: CPA、CPI、パートナーシップ
- 参照文書: Doc-MS-003（デジタルマーケティング戦略）

【Aggregator】（英語: Aggregator）
- 読み: あぐりげーたー / アグリゲーター
- カテゴリ: 予約・流通
- 定義: 複数の供給元から情報・在庫を集約し、一元的に提供するプラットフォーム。メタサーチとも呼ばれる。
- 文脈: TripTripは将来的にアグリゲーター機能を強化し、複数OTAの価格比較を提供予定。
- 関連用語: メタサーチ、OTA、GDS
- 参照文書: Doc-CS-002（競争ポジショニング）

【Allotment】（英語: Allotment）
- 読み: あろっとめんと / アロットメント
- カテゴリ: 予約・流通
- 定義: ホテルや航空会社が旅行会社やOTAに事前に割り当てる在庫枠。確保在庫とも呼ばれる。
- 文脈: TripTripのホテルパートナーとの契約において、繁忙期のアロットメント確保が重要課題。
- 関連用語: 在庫管理、レベニューマネジメント、コミットメント
- 参照文書: Doc-OS-002（サプライチェーン管理）

【API Integration】（英語: API Integration）
- 読み: えーぴーあいいんてぐれーしょん / API連携
- カテゴリ: 予約・流通
- 定義: Application Programming Interfaceを通じた外部システムとのデータ連携。リアルタイム在庫・価格情報の取得に使用。
- 文脈: TripTripはHono/OpenAPIベースのRESTful APIを構築済み。外部GDS/OTAとのAPI連携を拡大予定。
- 関連用語: RESTful API、Webhook、GDS
- 参照文書: Doc-SA-002（システムアーキテクチャ）

### B

【B2B】（英語: Business to Business）
- 読み: びーつーびー
- 略語: B2B
- カテゴリ: 予約・流通
- 定義: 企業間取引。旅行業界では、OTAからホテル、航空会社への取引を指す。
- 文脈: TripTripの法人向けサービス展開、旅行代理店へのAPI提供。

【B2C】（英語: Business to Consumer）

- 定義: 企業から消費者への直接取引。TripTripの主要ビジネスモデル。
- 文脈: TripTripアプリを通じた個人旅行者への直接サービス提供。
- 関連用語: B2B、D2C、カスタマージャーニー

- 定義: オンラインで予約を受け付けるためのソフトウェアシステム。在庫管理、価格表示、決済処理を統合。
- 文脈: TripTripの予約エンジンはFlutterアプリ内に実装され、ホテル・チケット・レンタルの統合予約を実現。

- 読み: ぶっきんぐういんどう / 予約ウィンドウ

- 定義: 予約日から利用日までの期間。顧客の予約行動分析における重要指標。
- 文脈: TripTripでは訪日観光客の予約ウィンドウが平均45-60日と分析。早期予約割引施策の設計に活用。
- 関連用語: リードタイム、Last-minute booking、需要予測
- 参照文書: Doc-MA-004（市場トレンド分析）

- 文脈: TripTripのホテル予約における基本コミッション率は15%を想定。体験予約は20-25%。

- 定義: ホテルチェーンや航空会社が運用する集中型予約管理システム。
- 文脈: 大手ホテルチェーンとの連携においてCRS接続が必要。TripTripは将来的にCRS直接連携を計画。
- 関連用語: GDS、PMS、API連携
- 参照文書: Doc-SA-004（システム連携設計）

- 計算式: Gross Margin = (売上高 - COGS) ÷ 売上高 × 100
- 文脈: TripTripの粗利率目標は60-70%。
- 関連用語: COGS、営業利益率、EBITDA
- 参照文書: Doc-FM-002（財務モデル）

### O

【Operating Expenses】（英語: Operating Expenses）
- 読み: おぺれーてぃんぐえくすぺんしず / 営業費用
- 略語: OpEx
- カテゴリ: 会計
- 定義: 事業運営に必要な経費。人件費、マーケティング費、オフィス費等。
- 文脈: TripTripのOpExはR&D40%、S&M30%、G&A30%の配分を目標。
- 関連用語: COGS、営業利益、CapEx
- 参照文書: Doc-FM-002（財務モデル）

### P

【P/L Statement】（英語: Profit and Loss Statement）
- 読み: ぴーえるすてーとめんと / 損益計算書
- 略語: P/L
- カテゴリ: 財務諸表
- 定義: 一定期間の収益と費用を示し、利益を算出する財務諸表。
- 文脈: TripTripの月次P/Lで収益性を追跡。
- 関連用語: B/S、C/F、財務諸表
- 参照文書: Doc-FM-002（財務モデル）

### R

【ROI】（英語: Return on Investment）
- 読み: あーるおーあい / 投資利益率
- 略語: ROI
- カテゴリ: 財務指標
- 定義: 投資に対するリターンの割合。
- 計算式: ROI = (利益 - 投資額) ÷ 投資額 × 100
- 文脈: TripTripの各マーケティング施策のROIを測定・最適化。
- 関連用語: ROAS、IRR、投資回収
- 参照文書: Doc-FM-003（投資評価）
